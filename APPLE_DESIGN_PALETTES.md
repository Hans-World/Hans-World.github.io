## Palette 1: Space Gray Professional (ACTIVE)

### Design Intent
Professional, sophisticated, technical. Perfect for data scientists.

### Color Values
```css
--color-bg-primary: #1d1d1f;      /* Deep charcoal */
--color-bg-secondary: #2c2c2e;    /* Card backgrounds */
--color-bg-tertiary: #3a3a3c;     /* Elevated surfaces */
--color-primary: #0071e3;         /* Apple Blue */
--color-text-primary: #ffffff;    /* 18.1:1 contrast */
--color-text-secondary: #a1a1a6;  /* 7.8:1 contrast */
--color-text-tertiary: #6e6e73;
--color-success: #30d158;         /* Green */
--color-error: #ff3b30;           /* Red */
--color-warning: #ff9500;         /* Orange */
```

### Use Cases
- Data scientist portfolios
- Professional contexts
- Enterprise applications
- Tech companies

### Accessibility
- All colors: WCAG AAA compliant
- White on primary background: 18.1:1 contrast

## How to Switch Palettes

1. Open `styles/colors.css`
2. Replace `:root` variables with desired palette
3. Save file
4. No other changes needed
